## Arguments in `read_csv`

Note that in class we reviewed the various arguments read\_csv could
take, e.g. skip, na, etc. Review these!

Let’s try specifying column types:

``` r
litters_data = read_csv(file = "Data/FAS_litters.csv",
  col_types = cols(
    Group = col_character(),
    `Litter Number` = col_character(),
    `GD0 weight` = col_double(),
    `GD18 weight` = col_double(),
    `GD of Birth` = col_integer(),
    `Pups born alive` = col_integer(),
    `Pups dead @ birth` = col_integer(),
    `Pups survive` = col_integer()
  )
)
tail(litters_data)
```

There are a bunch of reasons to use `read_csv` rather than `read.csv`
because of all kinds of things that happen in the background. `read.csv`
doesn’t give a tibble, for one.

## How do I export data?

Instead of reading data in, we’re going to write data out. Again, use
`write_csv` rather than `read_csv`.

``` r
write_csv(fellow_df, "Data/fellowship_words.csv")
```
